# 项目目录

这里存放完整的实战项目。

## 项目结构

每个项目应该包含：
- `README.md` - 项目说明
- `Cargo.toml` - 项目配置
- `src/` - 源代码
- `docs/` - 项目文档（可选）

## 项目建议

根据学习进度，可以尝试以下项目：

1. **CLI 工具** - 文件处理、文本搜索等
2. **Web API** - RESTful API、GraphQL 等
3. **网络服务** - TCP/UDP 服务器、代理等
4. **系统工具** - 监控、日志收集等
5. **游戏** - 简单的命令行游戏

## 贡献

欢迎提交你的项目！请确保：
- 代码质量良好
- 有清晰的 README
- 遵循 Rust 最佳实践
