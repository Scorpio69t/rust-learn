# 代码示例目录

这里存放学习过程中的代码示例和练习。

## 目录结构

```
examples/
├── Chapter01/          # 第一章示例
│   ├── hello_cargo/    # Hello Cargo 示例
│   ├── guessing_game/  # 猜数字游戏
│   └── minigrep/       # 迷你 grep 工具
└── ...
```

## 运行示例

进入对应的示例目录，使用 `cargo run` 运行：

```bash
cd examples/Chapter01/guessing_game
cargo run
```

## 学习建议

- **不要只看代码**，要自己手写一遍
- **修改代码**，看看会发生什么
- **遇到编译错误**，仔细阅读错误信息，这是学习的好机会

**通过示例代码，加深对 Rust 概念的理解！** 💡
