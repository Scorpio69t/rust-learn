# 贡献指南

欢迎为这个 Rust 学习仓库做出贡献！

## 如何贡献

### 1. 报告问题

如果发现文档中的错误、代码示例的问题，或者有改进建议，请提交 Issue。

### 2. 提交改进

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request

### 3. 代码规范

- 遵循 Rust 官方代码风格（使用 `rustfmt`）
- 使用 `clippy` 检查代码
- 确保代码可以编译通过
- 添加必要的注释

### 4. 文档规范

- 使用 Markdown 格式
- 代码示例要完整可运行
- 添加必要的说明和注释
- 保持文档结构清晰

## 贡献方向

- 📝 完善文档内容
- 🐛 修复错误
- 💡 添加新的示例代码
- 📚 补充学习资源
- 🌟 改进文档结构

感谢你的贡献！🙏
