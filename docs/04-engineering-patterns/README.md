# 第 4 章：从"能写"到"写得专业"

Rust 难的地方不是基础，而是工程模式。这一章将帮助你掌握 Rust 的工程实践。

## 核心内容

### 1. [模式匹配 & Enum](./pattern-matching.md) ⭐⭐⭐
**Rust 的灵魂**

大量 C++/Go 哲学将在此被颠覆：
- `match` 表达式
- `if let` 和 `while let`
- 枚举的威力

### 2. [迭代器链](./iterators.md) ⭐⭐
**函数式思想**

- `map` → `filter` → `collect` → `flatten`
- 链式调用
- 惰性求值

### 3. [错误处理](./error-handling.md) ⭐⭐⭐
**Rust 的错误处理哲学**

- `Result<T, E>`
- `anyhow` - 快速错误处理
- `thiserror` - 自定义错误类型
- `?` 运算符的使用

### 4. [内存模型](./memory-model.md) ⭐⭐
**深入理解 Rust 的内存管理**

- `Box`（堆分配）
- `Rc`/`Arc`（引用计数）
- `RefCell`/`Mutex`/`RwLock`（内部可变性）
- `Pin`（高阶用法）

## 学习建议

- **这一章是进阶内容**，需要扎实的前置知识
- **多写代码**，理解每个模式的使用场景
- **阅读标准库源码**，学习最佳实践

## 预计时间

- 1-2 周（边学边实践）

---

**掌握这些工程模式，你就能写出专业级的 Rust 代码！** 💼
