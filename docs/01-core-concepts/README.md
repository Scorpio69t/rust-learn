# 第 1 章：四大核心概念

这是 Rust 最核心、最重要的四个概念。**建议按顺序学习，不要跳过任何一个。**

## 学习顺序

1. **[Ownership（所有权）](./ownership.md)** ⭐⭐⭐
   - 哪个变量拥有数据？
   - 离开作用域就 drop
   - 理解移动（move）语义

2. **[Borrow & Reference（借用与引用）](./borrowing.md)** ⭐⭐⭐
   - 可变引用唯一
   - 不可变引用可以有多个
   - 可变与不可变不能同时存在

3. **[Lifetimes（生命周期）](./lifetimes.md)** ⭐⭐
   - 生命周期不是"写给 Rust 看"的
   - 是告诉编译器"我保证这个引用不会悬空"
   - 理解生命周期标注

4. **[Trait（特征）](./traits.md)** ⭐⭐
   - 类似 C++ 的概念类 + 接口 + 类型约束
   - Rust 的多态机制
   - 理解 trait bound 和泛型

## 学习建议

- **每个概念都要写代码验证**
- **遇到编译错误不要慌，仔细读错误信息**
- **Borrow Checker 的错误信息非常详细，是学习的好材料**

## 预计学习时间

- 有 C++/Go 经验：3-5 天
- 新手：1-2 周
